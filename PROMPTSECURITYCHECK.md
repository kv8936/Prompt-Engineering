# Prompt for security analysis of code

## Prompt 1: 

```text
Implement rate limiting on all endpoints - max 5 attempts per 15 min on login routes.'
```
## Prompt 2: 

```text
Scan my entire codebase for hardcoded API keys, tokens, or passwords.
```
## Prompt 3:
```text
Move all sensitive data to environment variables - nothing exposed in the frontend or committed to Git.
```
## Prompt 4:
```text
Sanitize all user inputs and reject anything oversized or malformed.
```

## Prompt 5:
```text
Run a full security audit and report any remaining vulnerabilities.
```
